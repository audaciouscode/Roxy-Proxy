Roxy Proxy Setup

iPad

  1. This setup can only be completed when connected to a the WiFi network. Each WiFi network used by the iPad must be setup in the same way.
  2. Go to the Settings App.
  3. Choose WiFi.
  4. Tap the blue arrow next to the WiFi network you are using
  5. Near the bottom of the screen are the HTTP Proxy settings. Choose "Auto" and type in
    {{ user.profile.pac_url }}
  6. Exit the settings app. Roxy's login screen will pop up when using Safari. You may need to login to Roxy via Safari to continue using the internet on the iPad.
  7. Bookmark the Change Session Type page for easy access to control your session type at any time.
  8. Print these instructions in case you need to access them offline.
  9. To disconnect from Roxy, go through the setup instructions, but instead of adding the proxy address ({{ user.profile.pac_url }}), remove it.
