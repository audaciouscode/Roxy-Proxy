הגדרת רוקסי פרוקסי

פיירפוקס לחלונות

  1. פתח את פיירפוקס
  2. בתפריט הראשי בראש החלון לך ל: אפשרויות>אפשרויות >מתקדם> רשת> הגדרות > URL לתצורה אוטומטית של שרת מתווך. העתק והדבק או הקלד
    {{ user.profile.pac_url }}

  3. בחר אישור לשמירת השינויים וצא מחלון ההגדרות
  4. לחץ כאן על מנת לבדוק האם ההתקנה הושלמה
  5. הוסף את שנה סוג גלישה למועדפים על מנת להקל על הגישה והשליטה שלך בסוגי הגלישה האפשריים ברוקסי

הגדר דפדפנים אחרים

אם אתה משתמש בדפדפן אחר כמו אינטרנט אקספלורר אתה צריך להגדיר את רוקסי גם בו. לך להוראות התקנה.

בעיות נפוצות

הבעיה הנפוצה ביותר היא טעות הקלדה בקישור של רוקסי. נסה לסמן את כתובת הפרוקסי הזו: {{ user.profile.pac_url }}ללחוץ על control+c על מנת להעתיק אותה, ואז לעבור להגדרות LAN וללחוץ control+v על מנת להדביק אותה.

Roxy Proxy Setup

Windows Firefox

  1. Open Firefox.
  2. On the main menu at the top of Firefox go to: Tools > Options > At the top of the options window there is an Advanced tab > Network tab > Settings > Automatic proxy configuration URL. Copy and paste or type:
    {{ user.profile.pac_url }}

  3. Choose OK to save the changes and exit the options window.
  4. Click here to check if the setup is complete.
  5. Bookmark the Change Session Type page for easy access to control your session type at any time.

Setup Other Browsers

If you use another web browser such as Internet Explorer you need to setup Roxy to work with that browser as well. Go back to setup instructions.

Troubleshooting

The most common setup mistake is a mistyped pac file url. Try highlighting the proxy address, {{ user.profile.pac_url }}, pressing control+c to copy it and when in the LAN Settings press control+v to paste it instead of typing it.
