Roxy Proxy Setup

Thank you for beta testing Roxy! You may find it useful to have Roxy active in one browser but not another so that you can test to see if Roxy is causing an issue you are experiencing online or not, and so that you can browse Roxy-free without changing your system proxy settings or changing your session type. The instructions below show you how.

Advanced Beta Testing

Do you mainly brows the web with FireFox (FF) or another browser? If FF, follow FF main instructions, if non-FF, go to FF secondary instructions.

FF Main

If FF is your main browser, you can set only FF to use the proxy, and when you encounter an issue you think might be caused by the proxy, try another browser.

  1. Remove the proxy setting from your computer.
  2. In Mac: Open FF, go the the FireFox menu > Preferences > Advanced Tab > Network Tab > "Settings" button > Choose the very last option, "Automatic proxy configuration URL" and put the Roxy PAC url in:
    http://roxy.roxyproxy.org/proxy/logging-proxy.pac
    > Choose OK > Close the preferences window.

    In Windows: Open Firefox.On the main menu at the top of Firefox go to: Tools > Options > At the top of the options window there is an Advanced tab > Network tab > Settings > Choose Automatic proxy configuration URL. Copy and paste or type:
    http://roxy.roxyproxy.org/proxy/logging-proxy.pac
    Choose OK to save the changes and exit the options window.
  3. Make sure Roxy is working in FF: Test

FF Secondary

If FF is your secondary browser, you can set your other browsers to use Roxy, but have FF not use Roxy.

  1. Make sure Roxy is working in your main web browser: Test
  2. In Mac: Open FF, go the the FireFox menu > Preferences > Advanced Tab > Network Tab > "Settings" button > Choose "No Proxy" > Choose OK > Close the preferences window.

    In Windows: Open Firefox.On the main menu at the top of Firefox go to: Tools > Options > At the top of the options window there is an Advanced tab > Network tab > Settings > Choose "No Proxy" Choose OK to save the changes and exit the options window.
  3. You can test to see that Roxy is not active in FF: Test
