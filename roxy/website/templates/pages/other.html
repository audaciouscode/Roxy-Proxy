הגדרות רוקסי פרוקסי

אחר

רוב המכשירים המתחברים לאינטרנט מאפשרים לך להגדיר שרת פרוקסי. כל דפדפן שתומך בשרת פרוקסי יכול להשתמש ברוקסי. אתה יכול לחפש ברשת עזרה בהגדרות שרת פרוקסי למכשיר שלך ולהשתמש בקישור הזה על מנת להגדיר את שרת הפרוקסי או את קבצי ה PAC {{ user.profile.pac_url }}

  1. לחץ כאן לבדוק האם ההתקנה הושלמה
  2. הוסף את שנה סוג גלישהלמועדפים על מנת להקל על הגישה והשליטה שלך בסוגי הגלישה האפשריים ברוקסי
  3. הדפס את ההוראות הללו למקרה שתצטרך להתחבר בלי גישה לאינטרנט
  4. על מנת להתנתקמרוקסי, עקוב אחרי צעדי ההתקנה, אך במקום להוסיף את הכתובת של רוקסי,({{ user.profile.pac_url }}), הסר אותה.

Roxy Proxy Setup

Other

Most devices that connect to the web allow you to specify proxy settings. Any web browser that supports a proxy server can use Roxy. You can search online for help setting up a proxy on your device and use this URL for the automatic proxy configuration or PAC file: {{ user.profile.pac_url }}

  1. Click here to test if the setup is complete.
  2. Bookmark the Change Session Type page for easy access to control your session type at any time.
  3. Print the instructions you use to setup your device or browser in case you need to access them offline.
  4. To disconnect from Roxy, go through the setup instructions, but instead of adding the proxy address ({{ user.profile.pac_url }}), remove it.
