הגדרת רוקסי פרוקסי

אינטרנט אקספלורר

  1. פתח את הדפדפן
  2. בתפריט הראשי בראש החלון לך ל: אפשרויות אינטרנט>חיבורים>הגדרות LAN

  3. לחץ על השתמש ב script לקביעת תצורה אוטומטית והקלד :
    {{ user.profile.pac_url }}
    בטל את הסימון ב"אתר הגדרות באופן אוטומטי"

  4. סמן אישור ושמור הגדרות
  5. לחץ כאן על מנת לבדוק האם ההתקנה הושלמה
  6. הוסף את שנה מצב גלישה למועדפים על מנת להקל על הגישה והשליטה שלך בסוגי הגלישה האפשריים ברוקסי
  7. הדפס את ההוראות הללו למקרה שתצטרך להתחבר בלי גישה לאינטרנט
  8. על מנת להתנתק מרוקסי, עקוב אחרי צעדי ההתקנה, אך במקום להוסיף את הכתובת של רוקסי,({{ user.profile.pac_url }}), הסר אותה.

לאחר ההתקנה יכולה להופיע הודעה בדפדפן שאומרת "הגדרות אינטרנט מכובות כעת. הגדרות האינטרנט פחות בטוחות. לחץ על אפשרויות… רוקסי לא פוגעת באבטחת המחשב שלך. בחר "אל תציג הודעה זו שוב.

Roxy Proxy Setup

Windows Internet Explorer

  1. Open Internet Explorer.
  2. In het hoofdmenu bovenin het venster, ga naar: Extra> Internet Opties > Verbindingen tab > LAN Instellingen

  3. Vink "Automatisch configuratiescript gebruiken" aan en type in (kopieër en plak de volgende link):
    {{ user.profile.pac_url }}
    Dan, vink de optie "Instellingen automatisch detecteren" uit.

  4. Klik op OK om de instellingen op te slaan.
  5. Klik hier om te testen of de set-up compleet is.
  6. Voeg de Verander Sessie Type pagina toe als bladwijzer om makkelijk tussen sessie types te schakelen.
  7. Print deze instructies uit voor het geval dat u deze offline in wilt zien.
  8. Om het gebruik van Roxy te verbreken, ga terug naar de setup instructies, maar in plaats van het toevoegen van het proxy adres ({{ user.profile.pac_url }}), verwijder deze.

Een bericht kan verschijnen in de browser, “Intranet instellingen zijn nu uitgeschakeld. Intranet instellingen zijn minder betrouwbaar dan Internet instellingen. Klik hier voor opties...”. Roxy beinvloed uw computer’s veiligheid niet. Dit bericht kan daarom zonder risico worden genegeerd. Kies voor de optie “Laat dit bericht niet meer zien”.

Roxy Proxy Setup

Windows Internet Explorer

  1. Open Internet Explorer.
  2. On the main menu at the top of the window go to: Tools > Internet Options > Connections tab > LAN Settings

  3. Check "Use automatic configuration script" and type in:
    {{ user.profile.pac_url }}
    Then uncheck the item that says "Automatically detect settings"

  4. Choose OK to save the settings.
  5. Click here to test if the setup is complete.
  6. Bookmark the Change Session Type page for easy access to control your session type at any time.
  7. Print these instructions in case you need to access them offline.
  8. To disconnect from Roxy, go through the setup instructions, but instead of adding the proxy address ({{ user.profile.pac_url }}), remove it.

A message may appear on Internet Explorer that says, "Intranet settings are now turned off by default. Intranet settings are less secure than Internet settings. Click for options..." Roxy does not compromise your computer's security. This message can safely be ignored. Choose the option "Don’t show this message again"

Setup Other Browsers

If you use another web browser such as Firefox you need to setup Roxy to work with that browser as well. Go back to setup instructions.
