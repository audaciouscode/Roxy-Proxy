גוגל כרום לחלונות

דפדפן כרום משתמש באותן הגדרות כמו אינטרנט אקספלורר. כך שאם כבר עקבת אחרי ההוראות להתקנת רוקסי באינטרנט אקספלורר פשוט לחץ כאןעל מנת לבדוק האם רוקסי פעילה

  1. פתח את כרום
  2. בקצה הימני העליון של הדפדדפן הכנס ל תפריט ראשי> הגדרות (נפתח כלשונית נוספת)>הצג הגדרות מתקדמות> רשת > שנה הגדרות שרת > שנה הגדרות שרת proxy נפתח חלון נוסף באנגלית. תחת connection > LAN Settings > בטל את הסימון ב "Automatically detect settings".סמן "Use Automatic configuration script" > הדפס :
    {{ user.profile.pac_url }}
  3. לחץ על אישור לשמירת שינויים
  4. לחץ כאן על מנת לבדוק האם ההתקנה הושלמה
  5. הוסף את שנה מצב גלישה למועדפים על מנת להקל על הגישה והשליטה שלך בסוגי הגלישה האפשריים ברוקסי

Roxy Proxy Setup

Google Chrome on Windows

The Chrome browser uses the same settings as Internet Explorer, so if you have already followed the instructions to setup Roxy with Internet Explorer just Click here to see if Roxy is active.

  1. Open Chrome.
  2. On the top right of your browser, just below the red X, there is a button with three lines on it. Click this button > Settings > At the bottom, click "Show advanced settings" > Under "Network choose "Change proxy settings" > Choose the "Connections" tab > LAN Settings > Uncheck "Automatically detect settings". Check "Use Automatic configuration script" > Type in or paste:
    {{ user.profile.pac_url }}
  3. Choose OK twice to save the changes.
  4. Click here to test if the setup is complete.
  5. Bookmark the Change Session Type page for easy access to control your session type at any time.

Setup Other Browsers

If you use another web browser such as Firefox you need to setup Roxy to work with that browser as well. Go back to setup instructions.
